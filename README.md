# LesVerbes

Большой офлайн-справочник и тренажёр по французским глаголам.

## Live

- GitHub Pages: <https://polpal1623.github.io/LesVerbes/>
- Репозиторий: <https://github.com/PolPal1623/LesVerbes>

## Что внутри

- 40 глаголов.
- 5 времён: `Présent`, `Imparfait`, `Passé composé`, `Futur proche`, `Futur simple`.
- Три основных режима: справочник, тренировка и статистика.
- Локальное сохранение прогресса в `localStorage`.
- Экспорт и импорт результатов в JSON.

## Запуск

1. Откройте `index.html` в браузере.
2. Проект не требует сборки и работает полностью офлайн.

## Редактирование

- База глаголов хранится в константе `VERBS`.
- Основные режимы и представления собраны в одном HTML-файле.
- Интерфейс дополнен ссылками на общую библиотеку French Learning Suite.
